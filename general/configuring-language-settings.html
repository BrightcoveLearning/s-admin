The Language Settings is where you define a Primary Language and then add and manage additional metadata languages. The Primary Language is the language your account is currently using for video metadata. Before adding additional languages to your account, your existing metadata will be associated with the language you define as the Primary Language. The Primary Language is what is initially displayed when you edit video properties in the Media module.

Setting a Primary Language

To set the Primary Language:

  1. Login to {{ site.product }}.
  2. Open the Admin module.
  3. Click Language Settings. Note that you must be an account administrator to access to the Admin menu.
  4. Click Primary Language and enter your primary language. As you type, a list of matching languages appears.
  5. Click on a language to select it.
  6. Click Save Language.
To change the Primary Language, click on the language, select a new language and then click Save.

Adding additional languages

After a Primary Language is saved, additional metadata languages can be added. By adding additional languages, you will be able to enter additional, multi-lingual metadata in the Media module. To add additional languages, follow these steps.

  1. Click + Add Language.
  2. Enter your language. As you type, a list of matching languages will appear.
  3. Click on a language to select it.
  4. Click Save Language. The language will be added to the Additional Languages table.
  5. Click + Add Language to add additional languages as needed.

To edit the language name (as it will appear in the Media module), click on it, make any changes and then click Save. To delete a language, click the delete icon ().

Note: If you delete a language that is in use, all metadata associated with the language will be hidden. To restore the associated metadata, add the language again.
Note: A maximum of 40 metadata languages can be added.
